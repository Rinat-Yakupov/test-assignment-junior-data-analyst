## Задание 1: Пони тоже кони

**Условие:**
Вас просят разработать модель, классифицирующую лошадок и пони. Вместо
разработки вы нашли на GitHub две интересные модели и после прогона на ваших
данных одна из них показала ROC-AUC=0.7, а другая ROC-AUC=0.1. Какую модель вы
возьмете для дальнейшей работы и что будете с ней делать? 

**Ответ:**

Для дальнейшей работы я возьму модель с ROC-AUC = 0.1. Такое значение означает, что модель систематически путает классы местами. Значит, нужно инвертировать выход данной модели, т.е. вычесть из 1. Получим ROC-AUC = 1 - 0.1 = 0.9, что лучше чем модель с ROC-AUC = 0.7.



## Задание 2: Ручной счёт ROC_AUC

**Условие:**
Классификатор выдал следующие прогнозируемые метки класса и вероятности
принадлежности к классу "1". На основе полученных данных рассчитайте метрику
ROC_AUC. Тезисно описать ход решения.

**Ход решения:**

Показатель ROC_AUC рассчитал с помощью формулы AUC через статистику Манна_Уитни.
1) Сначала проранжировал все объекты по оценке вероятности, по возрастанию. Строка с оценкой вероятности 0,25 получила ранг 1.
Строка с оценкой вероятности 0,95 получила ранг 15.
2) Просуммировал ранги строк с истинной меткой класса 1. R_pos = 70.
3) посчитал сколько объектов с истинной меткой 1 и истинной меткой 0. n_pos = 7, n_neg = 8.
4) Подставил получившиеся значения в формулу, получилось ROC_AUC = 0,75.


**Ответ:** 0,75


## Задание 3: Ручной счёт корреляции

**Условие:**

Рассчитайте линейную корреляцию Пирсона, на основе данных.

Какой вывод можно сделать на основе полученного результата? Можно ли утверждать, что
существует причинно-следственная связь между количеством чашек кофе, выпитых
студентами в течение экзаменационного дня, и их итоговым баллом за экзамен?

**Ход решения:**
1) Рассчитал среднее по каждой переменной: среднее число чашек кофе, среднее по баллам.
2) Посчитал отклонения каждого значения от среднего по обеим переменным и их попарные произведения.
3) Применил формулу коэффициента корреляции Пирсона: сумма произведений отклонений, делённая на корень из произведения сумм квадратов отклонений.
4) Получил r ≈ -0,85.


**Ответ:** Коэффициент корреляции равен -0,85, что говорит о наличии отрицательной взаимосвязи между количеством чашек кофе, выпитых студентами и их итоговым баллом. При этом, нельзя утверждать, что существует причинно-следственная связь между количеством выпитых чашек и итоговым баллом. Возможна третья, скрытая переменная, которая и является "истинной" причиной, например 
отсутствие недосыпа.
